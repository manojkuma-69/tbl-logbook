Firebase Setup — Connect Cloud Sync

⚡ One-Time Setup — Takes 5 Minutes

01
Go to console.firebase.google.com and sign in with your Google account.
02
Click "Add Project" → name it tbl-logbook → disable Google Analytics → click "Create Project".
03
Inside your project, click the </> Web icon → register app with nickname tbl-logbook → copy the firebaseConfig object shown.
04
In the left sidebar go to Build → Firestore Database → click "Create database" → choose "Start in production mode" → pick a server location closest to India (e.g. asia-south1).
05
In Firestore go to Rules tab and replace the rules with:
rules_version = '2';
service cloud.firestore {'{'} match /databases/{'{database}'}/documents {'{'} match /users/{'{userId}'}/trades/{'{trade}'} {'{'} allow read, write: if request.auth != null && request.auth.uid == userId; {'}'} {'}'} {'}'}
Click Publish.
06
Go to Build → Authentication → click "Get started" → enable Google sign-in provider → add your domain (e.g. yourusername.github.io) under Authorized domains.
07
Open your index.html file in a text editor → find the firebaseConfig section near the top → replace each PASTE_YOUR_... value with the values from step 03 → save the file → re-upload to GitHub replacing the old index.html.
08
Visit your GitHub Pages link, click "Continue with Google", sign in — your trades will now sync across every device instantly. ✓
